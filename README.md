# PrivateWindowCheck
**Demo:** https://jlynx.github.io/PrivateWindowCheck/


Javascript used to check weather a users browser is in private mode or not


This works on:

**Windows**
* Chrome
* FireFox
* Edge
* IE(11)

**OS X**
* Safari

**Android**
* Chrome
* Firefox
* Opera

**iOS**
* Safari


If the browser is not listed, then I have not tested it
